## 🚀 Como rodar qualquer projeto

### Pré-requisitos

### Passos

```bash
# 1. Clone o repositório
git clone https://github.com/Waidemannm/front-testes-apis

# 2. Entre na pasta do projeto desejado
cd PastaDesejada

# 3. Instale as dependências
npm install

# 4. Rode em modo desenvolvimento
npm run dev
```

Acesse o localhost no browser.
